# 🎮 GameHub

Uma plataforma web interativa para descoberta e exploração de jogos, com integração em tempo real com a API da RAWG.

---

## 🚀 Sobre o Projeto

O **GameHub** é uma aplicação front-end desenvolvida com foco em experiência do usuário, permitindo buscar, filtrar e visualizar jogos de forma dinâmica e intuitiva.

A aplicação consome dados da API da RAWG e apresenta uma interface moderna, com animações suaves, efeitos visuais e navegação fluida entre páginas.

---

## ✨ Funcionalidades

* 🔍 Busca em tempo real por jogos
* 🎯 Filtro por categorias (Ação, RPG, Estratégia, FPS)
* ⭐ Sistema de favoritos com persistência (LocalStorage)
* 📦 Carregamento dinâmico de jogos (paginação)
* 🪟 Modal interativo com informações rápidas
* 📄 Página de detalhes completa de cada jogo
* 🎨 Interface moderna com animações e efeitos visuais
* ⚡ Transições suaves entre páginas

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (Flexbox, Grid, Animações, Variáveis CSS)
* JavaScript (ES6+)
* API REST (RAWG)
* LocalStorage

---

## 🧠 Aprendizados

Neste projeto, foram aplicados conceitos importantes como:

* Consumo de API externa com `fetch` e `async/await`
* Manipulação dinâmica do DOM
* Gerenciamento de estado no front-end
* Boas práticas de organização de código
* Criação de interfaces responsivas e interativas

---

## 🎨 Diferenciais

* Interface com efeito **Glassmorphism**
* Animações suaves com `@keyframes`
* Efeito 3D interativo nos cards
* Skeleton loading (carregamento elegante)
* Sistema de favoritos persistente
